# Optimum Tasks

Given list of tasks with different start time, end time, and weight/score, find the most optimum subset of tasks without time overlap

## Getting Started

### Prerequisites

- Node JS
- VS Code (recommended)

### Installing

```
npm install
```

```
npm run start
```

## Running the tests

```
npm run test
```
